This project contains the administration area which is included in Griffin.MvcContrib.

The administration area can be used in your web project to handle user accounts (MembershipProvider) and to
manage all localizations (views, validations and models) in your web project.

---

All files are linked in Griffin.MvcContrib. Everything is in a seperate project to make it easier to develop 
and test the administration area.

---

All you  need to do to include the admin area is the following:

Nuget: 
install-package griffin.mvccontrib
